Test Formatting

There are three main types of test formatting. Those are: Bold, Italics, and Underline.

Inline vs. Block Elements

A block element moves the content to a new line. The paragraph and header elements are block elements. An Inline element does not display its content to a new line. It keeps the content on the same line as the previous content. Bold, italics, and underline are all inline elements.

Vertical Text Spacing

There are two elements that help with vertical spacing between block elements. A line break element adds an empty line and moves the content down to a new line. Its similar to the ENTER key in a Google/Word doc. A horizontal rule is a line break with a horizontal bar across the page. Both of these are block elements
