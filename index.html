Inline vs. Block Elements

A block element moves the content to a new line. The paragraph and header elements are block elements. An Inline element does not display its content to a new line. It keeps the content on the same line as the previous content. Bold, italics, and underline are all inline elements.

Images

An image element is an inline element which inserts pictures into the document. An image element is self-0closing, which means it does not have a closing tag. It has one major attribute, src, which is a filepath that points to the images' location. Those filepaths can either be absolute or relative.

Simple Image Example

Image Attributes

There are four attributes commonly used for images. Width and height are used to resize the images. Alt allows us to use text description of images for accessibility and broken images. Title allows a hover-over text to appear.

Width and Height

The width and height can be measured in many different ways, but more common ones are pixels and percentages. Using pixels, you set the specific, precise size based on the pixels. Using percentages, you set the dimensions based on a percentage of the entire page. In either case, setting the width alone respects aspect ratio, while setting both does not.

Alt Attribute

EVERY picture should at least have the alt attribute. The alt attribute helps with screen readers for visually impaired and in the cases that the image is broken.

Relative vs. Absolute

An absolute filepath includes all the navigation to get to the server, folder, and file. The examples above are all absolute filepaths. An absolute filepath is used when the image is stored on a different server or device than the HTML document.

A relative filepath only includes folders and files. The example below is a relative filepath. A relative filepath is used when the images are stored alongside the HTML document.

Lists

There are two main types of lists. An unordered list is a bulleted list of items. An ordered list is a numbered list of items. In an ordered list, the style of the list can be altered using attributes list-style-type, and type. You can also change the order and numbering of a ordered list by using the attributes start, reserved, and value.

An unordered list only has ONE attribute, list-style-type. It can have values of disc (default), circle, square, and none.

An ordered list has multiple attributes broken into two sets, The first set of attributes determines the type of numbering to use. The second set determines values for the items in the set.

The attribute type determine what numbering system to use. There are 4 different types:

  • type = 1, number (default)
  • type = A, uppercase letters
  • type = a, lowercase letters
  • type = I, uppercase Roman numerals
  • type = i, lowercase Roman numerals
